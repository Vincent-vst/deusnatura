## Install 

```bash
git clone https://github.com/vincent-vst/deusnatura.git
cd deusnatura
npm run dev
```

## Build and deploy

```bash
npm run build
serve -s dist
```

*or using the dockerized version*  

```bash
docker compose up --build
```
