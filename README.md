# Deus Natura 

Deus Natura is a music platform for musician wanting to improve their ear. 
Deus natura is live at : https://deusnatura.com 
This is a [React Native](https://reactnative.dev/) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app) 

## Installation  

1. Clone the repo and install the dependencies 

```bash
git clone https://github.com/vincent-vst/deusnatura.git
cd deusnatura
npm i
```

2. Run the app on the desired emulator

__on a web browser__ : `npm run web`
__on an android emulator__ : `npm run android`
__on an ios emulator__ : `npm run ios`
